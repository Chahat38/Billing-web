 # 🧾 Online Billing System (JavaScript Project)

## 📌 Overview

This is a simple **Online Billing System** built using **HTML, CSS, and JavaScript**.
It allows users to enter their details, calculate discounts, apply tax, and generate a final bill dynamically.

---

## 🚀 Features

* User-friendly interface
* Input validation (no empty or invalid values)
* Automatic discount calculation:

  * 20% for amount ≥ 5000
  * 10% for amount between 3000–4999
  * 5% for amount between 1000–2999
* Tax calculation (5%)
* Customer status:

  * Premium Customer
  * Regular Customer
* Auto form reset after calculation

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)

---

## 📂 How to Use

1. Enter **Consumer Name**
2. Enter **Reference Number**
3. Enter **Amount**
4. Click **Pay Bill**
5. View calculated bill with:

   * Discount
   * Tax
   * Final Amount
   * Customer Status

---

## ⚠️ Validation

* Name must not be empty
* Amount must be greater than 0
